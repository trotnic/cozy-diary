# Cozy


<p>Simple note-taking application</p>
<p>Created as a part of a task at <a href="https://rs.school">RSSchool</a></p>

<b>Used technologies:</b>
<ul>
  <li>CoreData</li>
  <li>RxSwift/RxCocoa</li>
  <li>RxDataSources</li>
  <li>Kingfisher</li>
  <li>Alertift</li>
  <li>MVVM + Coordinator</li>
</ul>
